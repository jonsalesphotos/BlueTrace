横切状态（采集中 / 暂停重连 / 已停止 + 运行日志）

专业采集工具，异常不发散。运行只有三种状态：🟢 采集中 · 🟡 暂停·重连中（连接断开 → 自动重连，原始 HEX 日志续写不丢数据）· 🔴 已停止（存储满等不可继续）。无法处理的坏包 / CRC 错 / 解码失败 / 重连记录一律进运行日志、不打断采集。进程被回收可恢复；运行中离开时采集 Tab 顶部常驻"正在采集"提示条。
