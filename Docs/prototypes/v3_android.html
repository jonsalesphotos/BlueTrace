BlueTrace · UI 原型 v3

第三版围绕多传感器 DUT 重构：一台待测设备同时输出 PPG / ECG / IMU / 地磁 / 温度 等多路信号， 手机端不仅采集，还能通过协议反向控制设备（开关传感器、设采样率、启停设备端算法）。 BLE 传输层显式区分低频组帧（多种低频量打一帧）与高频批包（单一高频量缓存 N 点批量发送）， 采集主屏支持波形视图 ⇄ 原始分包流视图双视图切换。视觉沿用 v1/v2 浅蓝 Material 3。

与设计文档对齐：DeviceRole / SensorProfile<T> / DeviceAssignment / MultiDeviceController / 会话级 manifest + monotonic 对齐锚点。
v3 在 v1/v2 基础上新增三条主线 ——（1）多传感器 DUT（PPG·ECG·IMU·地磁·温度）；（2）手机经协议反向控制设备（开关传感器 / 采样率 / 设备端算法）；（3）BLE 低频组帧 + 高频批包的打包分包可视化。术语与 Architecture / UI_Implementation 文档保持一致。
