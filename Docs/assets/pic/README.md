# 参考 App 截图 — Creek ACQ

本目录 20 张 `*_origin.jpg` 是**参考 App「Creek ACQ」**的界面截图，作为 BlueTrace 数据采集流程的**外部参照**留存。
**仅供设计/交互参考，不是 BlueTrace 本身的界面，也不是需求真源**（真源以 [`/SPEC.md`](../../../SPEC.md) + [`prototypes/v4_android.html`](../../prototypes/v4_android.html) 为准）。

## 来源 App 速记（Creek ACQ）

底部一级 Tab：`Creek ACQ` / `More`。核心采集流程与 BlueTrace 高度同构：

| 截图涉及 | 内容 |
| --- | --- |
| 首页 | Raw Data File · Device · User · ACQ Project（**Wear / Unwear**）· Start / Offline Collect |
| Device | BLE 设备列表（如 `TI_S4001`、`Polar Sense`），连接 / 断开 / Refresh |
| File | `*.rawdata` 原始数据文件列表（搜索 · 多选 · Delete / Send to） |
| User | 用户档案（性别 / 出生 / 身高 / 体重） |
| Chart Mode | 图表模式开关 |
