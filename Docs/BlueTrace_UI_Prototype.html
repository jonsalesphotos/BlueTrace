BlueTrace 数据采集 App · UI 原型

面向 BLE PPG 信号评测的 Android 数据采集应用原型。基于 Material 3 设计语言，主色取自仓库 theme/Color.kt 中的 primaryLight = #415F91；以单/双设备同模型为核心， 全程支持 1..N 角色并行采集，长按退出避免误触。本原型覆盖 14 个关键界面与 3 类弹层/系统入口。

Material 3 Compose 风格 多角色同模型 FGS 后台采集 长按退出

主色板（取自仓库主题）

primary / primary-container / tertiary / tertiary-container / error / success

角色色彩约定

DUT（待测设备）用 primary 蓝；REFERENCE（标准心率带）用 tertiary 紫红。同一会话中可单独或并存。

状态徽章

采集中 需要关注 已断开

退出方式

采集运行中按返回键 = 弹层；要真正停止采集需要 长按"停止采集"按钮 2 秒，避免误触。
