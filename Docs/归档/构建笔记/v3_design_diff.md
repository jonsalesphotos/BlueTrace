## AskUserQuestion 裁决（v3-C）

- **Q1（GNSS 入口）→ 用户裁决：设置页不应有此开关；GNSS 改为「进入采集时采集数据勾选即可」。**
  - 即：① 移除 设置→本机 GNSS 行（及其 GNSS A/B/C 子屏入口）；② GNSS 作为 采集类型（运行C / CollectType）里的一个勾选项，写 SessionConfig.gnssEnabled。
  - **本轮不实施**：第②半（采集类型勾选）会动采集管线 = 在途 GNSS WIP（§3 不卷入）；只做第①半会让 GNSS 不可达=半成品。→ **并入 GNSS WIP 一起落地**，不混进本轮纯视觉收敛提交。已记忆 [[gnss-relocation-decision]]。
- **Q2（性别文案）→ 用户裁决：照原型用英文枚举 Male/Female/Other（不本地化）。**
  - 已改 `sexLabel()` 返回英文常量。**有意偏离 SPEC §7.6**（枚举走资源本地化）——用户口径优先。提交说明里列明。
