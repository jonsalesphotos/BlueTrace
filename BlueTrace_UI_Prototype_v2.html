主色板

primary 蓝（DUT/主操作）· tertiary 紫（REFERENCE）· error · success。沿用 v1 的 Material 3 token 名，但向竞品的清新蓝青风格靠拢。

底部 Tab 出现规则

仅在 采集 / 文件 / 设置 三个主屏出现。所有子页（设备、用户、模式编辑、设置子页）顶部带返回按钮，不显示 Tab Bar。
数据采集页是独立全屏页，进入后不能切换到任何其他页面，必须长按 End 2 秒退出。

采集独占 + 长按 2 秒退出

采集运行中 Tab Bar 隐藏 / 系统返回键拦截 / Home 键暂停后台。结束按钮必须长按 2 秒（环形进度可视化），松手即取消，避免误触丢数据。

文件存储与导出

采集时先写入 App 私有目录：Android/data/com.bluetrace/files/session_xxx/；用户点"导出"再复制到 Download/BLEData/，或点"上传"经远程服务器发送。
